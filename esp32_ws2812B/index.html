Contrôle bandeau LED WS2812B avec une carte ESP32

LED SK6812

Caractéristiques principales :
- Adressable individuellement : Chaque LED WS2812B contient son propre circuit intégré (CI), ce qui permet de contrôler la couleur et la luminosité de chaque LED indépendamment.
- Plusieurs options de couleurs : Outre les couleurs rouge, vert et bleu (RGB).
- Type de LED : Généralement de type SMD 5050, ce qui signifie qu'elle mesure 5,0 mm x 5,0 mm.
- Communication : Utilise un protocole de communication série unipolaire, ce qui signifie qu'un seul fil de données est nécessaire pour contrôler plusieurs LED en cascade.
- Tension de fonctionnement : Généralement 5V ou 12V DC, selon le modèle.
- Intensité variable : Chaque couleur (rouge, vert, bleu) peut être modulée en intensité sur 256 niveaux (8 bits par couleur), offrant ainsi une large gamme de couleurs et d'effets de gradation.

Utilisation avec une carte ESP32

Il faut ajouter une bibliothéque à l'IDE Arduino pour utiliser ces LEDs. On pourrait coder sans blibliotéque suplémentaire mais le code serait complexe et difficilement lisible.

Nous allons la bibliothèque "ESP32_WS2812_Lib" à ajouter dans Arduino IDE. Ce travail nous simplifira le controle des LEDs.

Ajout de la bibliothéque (library) et ouvrir les exemples de code

Ajouter la bibliotèque dans l'IDE Arduino.

Code pour tester :

#include "ESP32_WS2812_Lib.h"

#define LEDS_COUNT 120
#define LEDS_PIN   2
#define CHANNEL    0

ESP32_WS2812 strip = ESP32_WS2812(LEDS_COUNT, LEDS_PIN, CHANNEL, TYPE_GRB);

void setup() {
  strip.begin();
  strip.setBrightness(0);
}
void loop() {
  for (int i = 0; i < LEDS_COUNT; i++) {
    setLedColorData(i,255,0,255);
    delay(20);
  }
  strip.show(); 
  delay(20);
}
